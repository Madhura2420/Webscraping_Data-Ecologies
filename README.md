# Data Ecologies

This project explores relationships between smart city infrastructures and ecological intelligence using web scraping, machine learning, generative AI, ComfyUI, and Processing.

Datasets:
- Pinterest smart city imagery
- Wikipedia smart city texts
- BBC Earth ecology imagery

Workflow:
Web scraping → Vectorisation → Clustering → GPT ecological translation → DALL·E/ComfyUI generation → Processing animation → Premiere Pro compositing

Software:
Python, OpenAI API, ComfyUI, Processing, Premiere Pro
